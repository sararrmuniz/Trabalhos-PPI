Ao comparar os arquivos "busca-endereco.php" dos três exemplos estudados, nota-se que no Ex3 é feito o carregamento da string JSON da requisição e a converção da string JSON em objeto PHP. Já analisando os arquivos "index.html" de todos os exemplos, nota-se que no Ex1 a string JSON é convertida em objeto JavaScript ,em seguida os dados retornados são utilizados para preencher o formulário. Já no Ex2, o objeto XMLHttpRequest é criado normalmente, porém a propriedade responseType é definida e, logo em seguida, a resposta é tratada caso ocorra erro. Por outro lado, no Ex3 o cabeçalho HTTP 'Content-Type' para envio de JSON é definido, os dados são utilizados para preeencher o formulário e, por fim, converte objeto JS para JSON e envia requisição.
